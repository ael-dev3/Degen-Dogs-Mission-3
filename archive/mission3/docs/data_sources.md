# Mission 3 Data Sources

## Primary source

- Base Mainnet JSON-RPC logs from the verified Mission 3 auction house.
- Contract state reads from the verified Mission 3 auction house.

## Verified contracts

Machine-readable verified config lives in `archive/mission3/config/mission3_contracts.verified.json`.

## Event sources

Event signatures and topic hashes live in `archive/mission3/config/mission3_events.verified.json`.

The archive currently indexes:

- `AuctionCreated(uint256,uint256,uint256)`
- `AuctionBid(uint256,address,uint256,bool)`
- `AuctionExtended(uint256,uint256)`
- `AuctionSettled(uint256,address,uint256)`

## Dashboard data

The current dashboard's generated data lives in:

- `generated/`
- `public/generated/`

Those files are generated by `scripts/build_dashboard.py` and are separate from the archive database.

## Reward estimate note

Reward estimate metrics in the repo come from a 141-Dog reward-basis snapshot and intentionally exclude WOOF Vault Bonus. They are dashboard estimates, not official reward accounting.
