# Per-Dog archive records

Generated per-Dog archive records live under:

- `archive/dogs/by-id/<dog_id>.json`
- `archive/dogs/manifest.json`

The files are generated by:

```bash
python3 scripts/build_unified_dog_index.py
# or
npm run archive:unified:index
```

Each JSON file contains the normalized unified record, chain-specific links, provenance, amount metadata, and available bid transaction hashes. Markdown pages are intentionally not generated for every Dog yet so the repo stays lightweight while Mission 1/2 data continues to be refined.

Do not create empty folders or official-looking summaries for Dogs whose archive data is not verified.
